# POTPOURRI

Multi-**P**eriod **O**p**t**imal **Po**wer Flow for Distrib**u**tion G**r**ids with Sto**r**age Appl**i**cation

## Etymology
Potpourri - piece of music composed from various popular smaller works or melodies

## Requirements:

    - Python & Conda
This package requires Python (3.7 or higher) and Conda. Miniconda includes a Python version, so this can be done in a single installation. Conda should then be update to the latest version via `conda update conda`. See [https://wiki.iaew.rwth-aachen.de/w/index.php/Python](https://wiki.iaew.rwth-aachen.de/w/index.php/Python) for more information on the installation of Python and Conda.

## Installation
**1. Install the environment:**
To install the environment described in the `environment.yaml`, run each line in the terminal:

    conda update conda
    conda env create -f environment.yml

In case you have already created an environment and just want to update it, replace 'create' with 'update' in the last line
